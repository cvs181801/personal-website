Weather App API

I had the opportunity to lead a group project with a couple of classmates who were also completing the Scrimba online course. One of the group members had to step away due to personal reasons, but the other teammate and I had fun completing this project together. I created the wireframe. He did the HTML, CSS, and added the API using vanilla Javascript (https://openweathermap.org). Then, I added the dynamic rendering for the weather information coming in from the resource. I also built the celsius and fahrenheit buttons, and added a feature that would allow the user to input their city to find weather info, in case their browser location service is turned off.
